Please create plan 06 in the plans folder for adding cards.

Add the cards from the Deuterium.ca.html file.

There should be a card partial. The card partial should take an icon, a colour, a title, content as input and a link.
Additionally there should be a new card grid layout created. This grid layout should take the number of colums as an input.

The Icon is an UTF-8 Icon and is optional.
The Title is optional
The colour should be applied as a class. It should be applied to the border, the icon and the title. The default colour is grey
The content should probably be yielded? If not please explain how the content should be passed into the partial.
The link is optional. The link should apply to the whole card. When hovered the whole card should glow.

The first application of cards should be for the footer. It should be updated to be a card.
